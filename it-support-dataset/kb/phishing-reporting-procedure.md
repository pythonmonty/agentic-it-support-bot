# Phishing Reporting Procedure

**Article ID:** KB-019

## Summary
How to report suspicious emails and what happens next.

## Symptoms
- Received a suspicious email or link
- Unexpected MFA prompt or credential request
- Email impersonating an executive or vendor

## Troubleshooting Steps
1. Do not click links or open attachments.
2. Use the Report Phishing button in Outlook.
3. Preserve the email for the Security team.

## Resolution Steps
1. Security triages the report and blocks malicious senders/URLs.
2. If credentials were entered, follow Password Reset + MFA Recovery.
3. User is notified of the outcome.

## Escalation Guidance
Confirmed compromise -> Security Incident Escalation.
