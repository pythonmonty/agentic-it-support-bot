# Access Certification Process

**Article ID:** KB-025

## Summary
Periodic recertification of user entitlements. Revoked-but-still-needed access is a common side effect.

## Symptoms
- Lost access right after a certification cycle
- Entitlement "revoked by certification"
- Access removed without a role change

## Troubleshooting Steps
1. Check the certification log for revoked entitlements.
2. Confirm whether the manager mistakenly declined an entitlement.
3. Verify the entitlement is still required for the role.

## Resolution Steps
1. Re-request and re-approve the wrongly revoked entitlement.
2. Refresh provisioning and validate access.
3. Document the correction.

## Escalation Guidance
Disputed revocations -> Compliance / Access Governance.
