# Document Management System Access

**Article ID:** KB-018

## Summary
Access to document libraries and records in the DMS.

## Symptoms
- Cannot open a document library
- "Access denied" on a specific folder
- Lost DMS access after a transfer

## Troubleshooting Steps
1. Confirm library-level permission group membership.
2. Check for record-class restrictions (regulatory libraries).
3. Verify provisioning sync.

## Resolution Steps
1. Add the user to the correct library permission group.
2. Re-validate access to the required folder/library.

## Escalation Guidance
Regulatory library access -> Compliance approval required.

## Related Systems
- Document Management System
- Identity Governance Platform
